Border Generator How-To

How to use the Border Builder

Preview Settings

This section is used strictly for the preview of the border.

Width & Height: The size of the preview window, scaled down to fit the page. This should be the width of whatever you would like to put the border behind.

Preview image: Upload an image of the scene you're going to put a border around to see how it looks!

Image Size: Only visible if you've uploaded an image. USe this to get an idea of how the border will look with different "thickness".

Background color: Change this to get a better idea of how the border will look on top of other scenes.

Static Settings

This section is the first part of your border's customization.

Border Radius: Set the roundness of the corners of the border.

Blur: Give your border a nice blurry effect.

Opacity: Set the transparency of the border.

Gradient Settings

The core of your borders look and feel

Gradient slider: Click on the slider to add a new point or select an existing point and drag it to get your desired gradient. With a point selected, use the color picker to set the color of that point!

Color picker: Set the color of the current point, don't forget to adjust opacity for some really awesome borders that loop around the frame with the spin animation. There are pre-defined colors you can access by click the square on the bottom left.

Gradient type: Select between the various gradient types.

Angle picker: Choose the angle of the linear or conic gradient

Circle center point: When the radial or conic gradient is selected, you can drag the center point of the circle on the preview window.

Animation Settings

Bring your border to life!

Spin: Probably the best animation to add, spins your gradient around the center point.

Blur: Slightly blurs and unblurs the border. *note: this will override the static blur.

Pulse: Shrinks and increases the border. *note: The shrink is relative to the canvas size.

Hue: Colorizes! *note: Only works on colors that aren't Blacks, whites or greys.

Cool, so what's next?

You've made the perfect border! Now what?

Add to OBS When you're done, simply click the text that says click to copy to get the url to paste as a browser source url in obs! I recommend setting the width and height in obs's browser source settings to the same as the values you set up under the static settings section. That's it, you're done!
